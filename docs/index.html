👤自己紹介

はじめまして、福岡県在住の河村麟太郎と申します。

おとどけ薬局に勤務しており、日々の業務を通じて医療・調剤の現場に携わっています。「学び続けることが好き」をモットーに、常に新しい知識やスキルの習得に取り組んでいます。

現在は Claude Code ワークショップ に参加し、AIを活用した開発手法を学習中です。業務の効率化・自動化にAIを役立てることを目指しており、薬局現場の課題をテクノロジーで解決することに興味を持っています。
